# Groq LLM Integration

This project demonstrates a basic integration of the Groq LLM API using Python.
It shows how to send chat-based prompts to a Groq-hosted large language model
and receive generated responses.

## Features
- Groq API integration
- Chat completions using LLaMA models
- Simple and beginner-friendly structure

## Requirements
- Python 3.8+
- Groq Python SDK
